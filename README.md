# End-to-End Data Pipeline Automation using Azure Data Factory

**Role:** Data Engineering Intern  
**Tools:** Azure Data Factory, Azure Databricks, Azure Blob Storage, Azure SQL, Power BI, Python, SQL  

## Project Overview
Implemented an automated ETL pipeline to ingest CSV datasets into Azure Blob Storage, transform them using Databricks (PySpark), load into Azure SQL Database, and visualize insights in Power BI.

## Architecture
1. Data source → Azure Blob Storage  
2. Azure Data Factory triggers Databricks job  
3. Databricks (PySpark) cleans and transforms data  
4. Data loaded into Azure SQL Database  
5. Power BI dashboards show results  

## Results
- Automated pipeline runs end-to-end  
- Cleaned and transformed data stored in Azure SQL  
- Interactive Power BI dashboards
